📊 Nykaa Performance Marketing Dashboard — Power BI

🔍 Project Overview
An interactive Power BI dashboard built to analyze Nykaa's digital marketing performance across platforms (Facebook, Google, Email). This dashboard helps marketing teams track revenue, conversions, acquisition costs, and campaign-level ROAS (Return on Ad Spend) in real time.

📌 Key KPIs Tracked

Metric                   Value
Total Revenue            ₹29 Billion
Acquisition Cost         ₹20.96 Million
Total Conversions        57 Million
ROAS                     1,366.96


💡 Key Insights

Google and Facebook are the top revenue-generating platforms, each contributing ~₹11.5bn
FB_Nykaa_Campaign had the highest conversions (23.1M) and revenue (₹11.52bn)
Email campaigns showed lower reach but maintained competitive ROAS (1,345.92)
Revenue trend shows strong performance Jul 2024–Mar 2025 with a sharp dip in Apr 2025 — indicating possible seasonal/budget shift
Total clicks across all campaigns: 260 Million+


🛠️ Tools & Technologies

Power BI Desktop — Dashboard creation
DAX — Custom measures for ROAS, revenue aggregations
Power Query — Data cleaning and transformation
Excel/CSV — Source data


🚀 How to Use

Download the .pbix file
Open with Power BI Desktop (free download from Microsoft)
Interact with slicers to filter by Platform and Campaign Name
Use the date range slider to analyze specific time periods

🎯 Business Use Case
This dashboard is designed for Performance Marketing Analysts at e-commerce companies to:

Monitor which ad platforms deliver the best ROI
Compare campaign-level performance at a glance
Identify revenue trends and seasonal patterns
Optimize budget allocation across channels
